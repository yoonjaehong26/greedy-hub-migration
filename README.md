# greedy-hub

그리디(Greedy) 개발 동아리 공식 허브 — 연혁·멤버·스터디·미션·게시판·지원·추억을 한곳에서 추적.

> 현재 단계: **기획(PRD)**. 코드 이전, 요구사항 정의 중.

## 문서

- [PRD (제품 요구사항 정의서)](docs/PRD.md) — 문제정의·기능·아키텍처·로드맵. _버전은 PRD 상단/부록 D 참고._

## 핵심 방향 (요약)

- **3계층**: 웹 = 기록(진실의 원천) / GitHub = 산출물 / Discord(봇) = 알림·액션.
- **프론트**: Next.js(SEO) · **백엔드**: 별도 언어 + REST/OpenAPI 계약 · **인증**: GitHub OAuth.
- 기존 `greedy-team/greedy-discord-bot`(리뷰어 매칭·구글폼 감시)은 재구현 없이 **통합**.
